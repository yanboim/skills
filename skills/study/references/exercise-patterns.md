# Exercise Patterns

Use this reference when selecting practice for the current study step.

## Practice Modes

### Recall

Use when the user just learned a concept and needs retrieval:

- define the term from memory
- list the steps without notes
- summarize the idea in two sentences

### Explain

Use when understanding matters more than rote memory:

- explain the concept to a beginner
- compare two similar ideas
- say why a wrong statement is wrong

### Apply

Use when the user should use the concept:

- solve one small problem
- classify an example
- pick the right approach for a case

### Debug

Use when misconception detection matters:

- fix a broken example
- find the wrong step
- predict what fails and why

### Transfer

Use when the user is ready for harder evidence:

- apply the concept to a new domain
- adapt the approach under a changed constraint
- explain how the idea differs in a similar scenario

## Selection Rules

- Start with recall or explain for fragile understanding.
- Use apply or debug once the basics are stable.
- Use transfer when the user is succeeding too easily.
- Keep one main exercise per turn unless the user asks for more.
